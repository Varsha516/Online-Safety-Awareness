 Cyber Awareness Website

This is a simple educational website built as part of a community service project to raise awareness about cyber threats and digital safety.

 Purpose

- Educate people about common cybercrimes
- Share cyber hygiene tips and prevention techniques
- Help users understand how to report cyber incidents
- Promote safe and responsible online behavior

** Pages Included**

- **cyber-hygiene.html** – Online safety habits and hygiene tips  
- **financial-frauds.html** – UPI scams, banking frauds, phishing, QR code scams  
- **malware-threats.html** – Spyware, ransomware, keyloggers and how to prevent them  
- **mobile-frauds.html** – Fake apps, excessive permissions, mobile security  
- **online-scams.html** – Online shopping scams, job offers, loan apps, crypto frauds  
- **social-engineering.html** – Fake police/tech support calls, impersonation  
- **help-and-support.html** – Helpline (1930), complaint portal, reporting steps  

 **Emergency Helpline**

- **Cybercrime Helpline:** 1930 (24x7)
- **Online Reporting:** [cybercrime.gov.in](https://cybercrime.gov.in)
- **Consumer Complaints:** [consumerhelpline.gov.in](https://consumerhelpline.gov.in)
- **RBI Phishing Email:** report.phishing@rbi.org.in

 Technologies Used

- HTML5  
- CSS3  
- No frameworks or backend – fully static  

##  Usage

You can view each `.html` file directly in the browser — no server required. Simply clone or download the repository.
